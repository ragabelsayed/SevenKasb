# SevenKasb
Seven Kasb is a mobile app for the admin to follow and do some main operations to his business.
## Fearures:
- [x] Beautiful UI & UX.
- [x] Work both online and offline. 
- [x] Animation.
- [x] Auto login
- [x] Screens:
    * Splash.
    * Login.
    * Home.
    * Expenses.
    * Add Purchase Bill.
    * Bill.
        - Purchase.
        - Sales.
    * Offline.
        - Purchase bill.
        - Extra expenses.
    * Inventory.
        - Daily.
        - Monthly.  
    * Settings.
        - User.
        - Password.
